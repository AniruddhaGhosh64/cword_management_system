# 🏟️ Smart Stadium – Frontend (React)

This is the frontend dashboard for the **Smart Stadium Crowd Management System**. It provides real-time visualization of stadium congestion, queue intelligence, and administrative controls.

## ⚙️ Features
- **Live Heatmap**: Real-time zone density tracking.
- **Smart Navigation**: Dynamic pathfinding UI.
- **Queue Intelligence**: Food stall wait-times and suggestions.
- **Admin Panel**: Live zone control and simulation overrides.
- **What-If Simulation**: Visualizing custom congestion spikes.

## 🛠️ Tech Stack
- **React (Vite)**
- **Vanilla CSS** (Custom variables, glassmorphism)
- **FastAPI Sync** (Polling mechanism)

## 🚀 Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Ensure backend is running at `http://localhost:8000`.
